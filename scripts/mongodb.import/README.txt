Все файлы должны с данными должны быть в кодировке UTF-8

country (Справочник стран)
_id : "de" - код страны ISO по классификатору (маленькими буквами)
n : { orig: "Deutschland", ru : "Германия", en : "Germany", de : "Deutschland" } - наименование страны на различных языках, orig является обязательным значением на языке оригинала. Код языка должен быть задан маленькими буквами.
pc : "+49" - код телефона страны
ico: "de_flag.png" - наименование файла с для страны. Наименование файла должно быть задано маленькими буквами, и образуется код страны ISO маленькими буквами, далее символ _ и наименование файла с раширением.

region (Справочник регионов мира с привязкой к странам)
cid : "ru" - код страны (маленькими буквами)
n : { orig : "Москва", ru : "Москва", en : "Moscow", de : "Moskau" } - наименование региона на различных языках, orig является обязательным значением на языке оригинала. Код языка должен быть задан маленькими буквами.

location_category (Иерархический справочник категорий мест)
_id : "garden" - идентификатор категории места (слово на английском языке уникальное в пределах справочника, маленькими буквами)
pid : "shops" - идентификатор родительской категории (маленькими буквами)
n : { ru : "Сад", en : "Garden" } } - наименование места на различных языках. Код языка должен быть задан маленькими буквами.

marital_status (Справочник семейных положений)
_id : "single" - идентификатор семейного положения (слово на английском языке уникальное в пределах справочника, маленькими буквами)
n : { ru: "Холост", en: "Single", de: "Aleinstehend" } - наименование на различных языках. Код языка должен быть задан маленькими буквами.
